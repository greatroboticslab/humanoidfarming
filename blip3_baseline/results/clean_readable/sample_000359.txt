Task: OFF-GRID GARDEN WATERING SYSTEM | Tank Set Up
Please break it into step-by-step subtasks.
------------------------------------------------------------
1)  Measure garden area to determine water needs.
2)  Select appropriate tank size based on water requirement.
3)  Choose location for tank away from direct sunlight.
4)  Ensure tank is level for proper water distribution.
5)  Install overflow pipe to prevent tank overfilling.
6)  Connect pump to tank using appropriate tubing.
7)  Position pump near water source for efficiency.
8)  Attach timer to control watering schedule.
9)  Install drip irrigation system to target specific plants.
10)  Test system for leaks and adjust as necessary.
