Task: Sprinkler System Install - Overview
Please break it into step-by-step subtasks.
------------------------------------------------------------
1) Assess yard size and layout for optimal placement of sprinklers.
2) Choose appropriate sprinkler type based on water pressure and coverage area.
3) Locate main water supply line and install necessary valves if needed.
4) Determine watering zones and mark areas to be covered by each sprinkler.
5) Dig shallow trenches along marked areas for sprinkler installation.
6) Attach sprinkler heads to underground PVC pipes using connectors.
7) Connect sprinkler heads to main water supply line through valves.
8) Test system for leaks and adjust settings as necessary.
9) Cover exposed pipe joints with soil or mulch for aesthetics and protection.
10) Program controller according to desired watering schedule and duration.
